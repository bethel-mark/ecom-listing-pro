# Amazon Listing 优化最佳实践

## 标题优化黄金公式

```
[品牌] + [核心关键词] + [特性/材质] + [规格] + [用途/场景] + [差异化卖点]
```

### 亚马逊A9算法关键要素
- 标题前 80 字符最重要（移动端截断点）
- 核心关键词出现在标题前段
- 禁止夸张词（Best, #1, Premium 等近年开始限制）
- 使用 Search Term 填充长尾词（后端关键词字段）

### 7大标题类型
1. **关键词堆叠型** — 适用于搜索驱动品类
2. **情感钩子型** — 适用于礼品/装饰品类
3. **问题解决型** — 适用于功能型产品
4. **对比型** — 适用于升级版/改进版产品
5. **场景型** — 适用于多用途产品
6. **规格主导型** — 适用于标准化产品
7. **品牌故事型** — 适用于品牌力强的产品

## 图片优化标准

### 亚马逊6张图标准
1. 主图：纯白背景，产品占85%画面，主体清晰
2. 尺寸图：标注精确尺寸，多角度展示
3. 功能图：核心卖点可视化
4. 细节图：材质/工艺特写
5. 场景图：使用场景展示
6. 对比图：与竞品或旧版对比

### 转化率优化要点
- 首图点击率决定流量入口
- 第2-3张图决定转化决策
- 信息图示比纯文字高 37% 转化率
- 使用对比色调突出核心卖点
