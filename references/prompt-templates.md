# Prompt 模板库

## 6张轮播图 Prompt 模板结构

### 模板 1 — Hero 主图
```
[3D风格] [视角] of [产品主体] floating above [底座/台面], [背景渐变], double soft shadow beneath, [左上标签] reading "[左信息]", [右下标签] reading "[右信息]", [顶部钩子/小标题], [中央大标题], [底部补充信息], [材质/光照说明], [禁止项]. Square 1:1 format, 2048×2048 px, [风格统一锚点].
```

### 模板 2 — 尺寸/对比
```
[3D风格] side-by-side comparison of [产品A] and [产品B] on [底座], [左侧描述], [右侧描述], central translucent glass info card listing "[规格信息]", subtitle "[对比标题]", bottom row "[工艺/品质信息]", [背景], [禁止项]. Square 1:1 format, 2048×2048 px, [风格统一锚点].
```

### 模板 3 — 材质故事
```
[视角] close-up of [材质特写] with visible [细节特征] on [背景色] background, frosted glass info card floating above with headline "[情感标题]", subtext "[材质说明]", footer "[品牌标语/情怀]", [光照], [风格]. Square 1:1 format, 2048×2048 px, [风格统一锚点].
```

### 模板 4 — 卖点网格
```
[排列方式] grid of [数量] frosted glass chips on [背景], each chip has [图标类型] and headline plus one-line explanation: [卖点1], [卖点2], [卖点3], [卖点4], bottom row "[总结标语]", [风格], [禁止项]. Square 1:1, 2048×2048, [风格统一锚点].
```

### 模板 5 — 场景三联
```
[3D风格] scene with [底座] holding [场景1] on the left, [场景2] in the center, [场景3] on the right, [产品] floating above, [场景标签], [中心标题], [背景], [光照]. Square 1:1, 2048×2048, [风格统一锚点].
```

### 模板 6 — 情感/礼物
```
[3D风格] scene of [产品] with [包装/礼物元素], [光照] from [方向], blurred [材质纹理] in background, top headline "[情感标题]", subtitle "[情感文案]", bottom footer "[产品保证]", [禁止项]. Square 1:1, 2048×2048, [风格统一锚点].
```

## 风格统一锚点

### 玻璃拟态风格
```
glassmorphism + soft 3D + isometric 3D cohesive style, [产品色系] color palette on [背景渐变], frosted glass panels with backdrop blur and 1px highlight edge, soft volumetric lighting
```

### 极简科技风格
```
minimalist clean style, soft gradient background, floating product, delicate drop shadows, precise typography, tech editorial aesthetic, no glare
```

### 手绘插画风格
```
hand-drawn editorial illustration style, soft pencil textures, watercolor accents, warm paper texture background, asymmetrical layout, artistic
```
